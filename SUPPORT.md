# Support

VALP is an open protocol draft. Use GitHub Discussions for questions, workflow
feedback, and runtime adapter exploration.

## Where To Ask

- Open-ended feedback and critiques:
  [Discussions](https://github.com/wcqxgjy6d8-pixel/Visible-Agent-Loop-Protocol/discussions)
- Runtime adapter compatibility:
  [Runtime adapter discussion](https://github.com/wcqxgjy6d8-pixel/Visible-Agent-Loop-Protocol/discussions/categories/ideas)
- Concrete bugs, broken examples, or unclear docs:
  [Issues](https://github.com/wcqxgjy6d8-pixel/Visible-Agent-Loop-Protocol/issues)
- Security-sensitive behavior:
  [SECURITY.md](SECURITY.md)

## Useful Details To Include

For protocol or runtime questions, include:

- the workflow you are trying to audit;
- which runtime or manual process is involved;
- what evidence the runtime can export;
- where "completed" can diverge from real task completion;
- the command, example, or task folder you tried.

Please remove private prompts, paths, screenshots, command output, and project
data before posting public examples.
