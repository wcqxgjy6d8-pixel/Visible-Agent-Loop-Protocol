# Privacy

The protocol is local-first.

Task evidence may contain:

- user prompts;
- project paths;
- file names;
- command output;
- agent reviews;
- screenshots or artifacts.

Users should treat `.herdr-loop/` as private project data unless they have
reviewed and sanitized it.

Skill recommendation backends must not upload task text or skill metadata unless
the user explicitly chooses a networked backend and approves that behavior.
