# Final Synthesis

Result: done.

Decision: keep VALP positioned as an early open protocol draft plus reference
CLI, with HERDR as the current public reference runtime and first-class
non-HERDR adapters still on the roadmap.

Disagreement: the critique that HERDR is closed source is not supported by the
current public repository evidence. The critique that VALP needs real cases,
clear adapter status, coverage transparency, and cautious Windows claims is
valid enough to document directly.

Evidence gap: this is a real Manual Mode documentation case study. It does not
prove live Full Mode dispatch, production deployment reliability, or a
first-class non-HERDR adapter.
